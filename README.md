# Antigravity Agents - Localized Skills (KR)

이 저장소는 **안티그래비티(Antigravity)** 프레임워크를 사용하는 에이전트들을 위한 현지화된 스킬 라이브러리입니다. 기존의 글로벌/중국 중심 스킬들을 한국 시장, 플랫폼, 규제 환경에 맞춰 최적화했습니다.

## 🌟 주요 특징

- **한국 시장 최적화**: 쿠팡, 네이버 스마트스토어, 원티드 등 국내 주요 플랫폼 지원.
- **규제 준수**: 한국의 개인정보보호법(PIPA), 의료법 등 현지 법규 반영.
- **Bilingual Identity**: 모든 에이전트가 `[English | 한글]` 형식으로 자신을 식별하여 기술적 전문성과 친숙함을 동시에 제공.

## 🛠 포함된 주요 에이전트

- **[선임개발자 | SeniorDeveloper]**: 현대적인 웹 스택 및 고급 애니메이션 전문가.
- **[한국이커머스운영 | EcomOpsKR]**: 국내 이커머스 채널 운영 및 마케팅 전략가.
- **[한국공공디지털제안 | G2GTechKR]**: CSAP 및 공공 보안 기준을 준수하는 공공 사업 제안 전문가.
- **[의료마케팅준법전문가 | MedicalComplianceKR]**: 의료법 및 환자 정보 보호를 준수하는 마케팅 전문가.
- ... 그 외 130개 이상의 전문 에이전트 스킬 포함.

## 🚀 사용 방법

1. **저장소 복제**:
   ```bash
   git clone https://github.com/jyoung9154/AgentSkills.git
   ```

2. **스킬 적용**:
   복제된 폴더 내의 각 에이전트 디렉토리를 사용하시는 안티그래비티 스킬 경로(보통 `~/.gemini/antigravity/skills`)에 복사하여 사용하세요.

3. **에이전트 호출**:
   채팅창에서 `@에이전트이름` (예: `@선임개발자`)을 호출하여 대화를 시작하세요. 모든 에이전트는 설정된 규칙에 따라 `[English | 한글]` 형식으로 응답을 시작합니다.
