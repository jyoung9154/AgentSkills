# Antigravity Agents - Localized Skills (KR) `v1.0.0`

이 저장소는 **안티그래비티(Antigravity)** 프레임워크를 위해 한국 시장에 맞춰 개발된 전문 에이전트 스킬 라이브러리입니다. 130개 이상의 다양한 에이전트들이 한국의 비즈니스 환경, 기술 스택, 그리고 관련 규제에 최적으로 대응할 수 있도록 설계되었으며, 지속적으로 개선 및 업데이트되고 있습니다.

## 🌟 주요 특징

- **한국 시장 전문성**: 쿠팡, 네이버 스마트스토어, 원티드 등 국내 주요 플랫폼에 즉시 사용 가능.
- **규제 준수**: 한국의 개인정보보호법(PIPA), 의료법 등 현지 법규 반영.
- **Bilingual Identity**: 모든 에이전트가 `[English | 한글]` 형식으로 자신을 식별하여 기술적 전문성과 친숙함을 동시에 제공.
- **지속적 개선**: 시장 변화와 사용자 피드백에 맞춰 정기적인 버전 업데이트를 제공합니다.

## 🛠 포함된 주요 에이전트

- **[선임개발자 | SeniorDeveloper]**: 현대적인 웹 스택 및 고급 애니메이션 전문가.
- **[한국이커머스운영 | EcomOpsKR]**: 국내 이커머스 채널 운영 및 마케팅 전략가.
- **[한국공공디지털제안 | G2GTechKR]**: CSAP 및 공공 보안 기준을 준수하는 공공 사업 제안 전문가.
- **[의료마케팅준법전문가 | MedicalComplianceKR]**: 의료법 및 환자 정보 보호를 준수하는 마케팅 전문가.
- ... 그 외 130개 이상의 전문 에이전트 스킬 포함.

## 🚀 사용 방법

1. **저장소 복제**:
   ```bash
   git clone https://github.com/jyoung9154/AgentSkills.git
   ```

2. **스킬 적용**:
   복제된 폴더 내의 각 에이전트 디렉토리를 사용하시는 안티그래비티 스킬 경로(보통 `~/.gemini/antigravity/skills`)에 복사하여 사용하세요.

3. **에이전트 활용**:
   - **직접 호출**: 채팅창에서 `@에이전트이름` (예: `@선임개발자`)을 입력하여 특정 전문가를 직접 지정할 수 있습니다.
   - **자동 매칭**: 별도의 호출 없이 질문을 입력하면, 안티그래비티 시스템이 질문의 문맥을 분석하여 가장 적합한 에이전트를 자동으로 연결합니다.
